# CloudSleep_JVer

**对应服务端已关闭**

云睡觉客户端（JVer），跨平台的那种   
云睡觉，很无意义的那种，你需要做的就是打开软件，然后躺下来睡觉。 

小麦云睡觉（发源地）：[https://github.com/WheatBox/WheatCloudSleep](https://github.com/WheatBox/WheatCloudSleep)

## 进入寝室后可以做什么？

睡觉  
睡觉  
睡觉  
咳咳，不开玩笑了  

左键点击 - 移动睡客（点在床上则会上床）  
左键拖动 - 移动视野  
鼠标滚轮 - 缩放视野（在手机端是双指缩放） 

## 指令
在聊天框以"/"开头  
/tp 玩家名称 *（传送到指定玩家旁边）*  
/animScaleFromFar *（展示一个镜头由远及近的动画）*
